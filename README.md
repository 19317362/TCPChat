TCPChat
=======

![alt tag](https://raw.github.com/Nirklav/TCPChat/master/screen.png)

Multi-user chat with mixed architecture (client-server, p2p)

Supports:
  1. Private messages.
  2. Rooms.
  3. Voice chat.
  4. Files sharing.
  5. Plugins (Example of simple plugin: https://github.com/Nirklav/ScreenshotPlugin).
